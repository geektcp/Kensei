Kensei
========================================
Kensei is a hack tool for WoW client, support Classic, TBC, Wotlk
<br/>
剑圣是服务于魔兽世界的国产高科技产品。
<br/>
轻量安全，绿色简单。
<br/>
持续更新，后续版本将支持1.12，2.43，3.33，3.32，3.35等资料片对应的客户端版本。
<br/>
